# osu-audio-compressor
Powershell files to automate re-encoding of mp3, wav, and ogg files in your osu! songs folder. Will not work for lazer. 
More specifically, all mp3 files are re-encoded to 128 kbps, ogg files are re-encoded with a quality of 3, and wav files are re-encoded to ogg files with a quality of 3. Only use these scripts if you want to decrease your songs folder size and don't really care about sound quality. Make sure to change the parent folder to match the location of your songs folder. 

Requires ffmpeg to be installed and available to the PATH
